Como conectar ao Google Sheets passo a passo

Abra sua planilha no Google Sheets e clique em Compartilhar (canto superior direito).
Em "Acesso geral", escolha Qualquer pessoa com o link e defina permissão como Leitor. Clique em Concluído.
Copie o ID da planilha da URL do navegador: docs.google.com/spreadsheets/d/[ID AQUI]/edit
Clique em Configurar no topo desta página, cole o ID e informe o nome exato das abas.
Clique em Salvar e conectar. Os dados serão carregados automaticamente.
Para atualizar com novos registros do formulário, clique em Sincronizar a qualquer momento.

Script Google Apps Script (opcional — atualização automática)

Cole este script no Apps Script da planilha para ordenar automaticamente por validade a cada envio do formulário:

function onFormSubmit(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet()
              .getSheetByName("Respostas ao formulário 1");
  var range = sheet.getDataRange();
  // Ordena pela coluna de validade (coluna D = índice 4)
  range.sort({column: 4, ascending: true});
}

Em Extensões → Apps Script, cole o código acima e adicione um gatilho em Acionadores → + Adicionar acionador: função onFormSubmit, evento Do formulário → Ao enviar formulário.
