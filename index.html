Como conectar ao Google Sheets

Abra a planilha AURA VENCIMENTOS no Google Sheets → Compartilhar → Qualquer pessoa com o link → Leitor
Copie o ID da URL: docs.google.com/spreadsheets/d/[ID]/edit
Clique em Configurar no topo, cole o ID e informe os nomes das abas (2025, BANCO DE DADOS, Respostas ao formulário 1)
Clique Salvar e conectar — os dados carregam automaticamente com preços e descontos
A aba Promoções mostra automaticamente todos os itens com status Aguarda com preço DE/POR e botão para gerar encarte
